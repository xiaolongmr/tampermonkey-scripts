## 脚本列表

### 1. 网页元素爬取工具
- **文件**: `爬取UI/网页元素爬取工具.js`
- **功能**: 可视化选择网页元素，提取HTML和CSS代码，支持实时编辑和预览
- **技术栈**: 原生JavaScript + TailwindCSS + Driver.js + Prism.js
- **特点**: 
  - 可视化元素选择
  - 实时代码编辑和预览
  - 新手引导功能
  - 支持TailwindCSS样式

### 2. 导航站数据404检测工具
- **文件**: `导航站数据404检测/导航站数据404检测.js`
- **功能**: 自动检测导航网站中收录链接的有效性，识别404等无法访问的链接
- **技术栈**: JavaScript + Axios + TailwindCSS
- **特点**:
  - 支持网站和图片检测
  - 并发请求处理
  - 详细的检测报告
  - 友好的用户界面

### 3. 花瓣去水印工具
- **文件**: `花瓣去水印/花瓣去水印.js`
- **功能**: 显示花瓣真假PNG、区分VIP素材和免费素材、花瓣官方素材去水印
- **技术栈**: 原生JavaScript + CSS
- **特点**:
  - 透明PNG检测功能
  - 自定义背景色区分素材类型
  - 官方素材去水印
  - 配置保存功能
