# 实现网站注册自动填写表单脚本

## 需求分析
- 目标网站：http://121.40.25.9:8080/register.html
- 功能：自动填写邀请码"1474728874"
- 表单使用Layui框架，邀请码输入框的name属性为"inviteCode"

## 实现方案
1. 创建新的UserScript文件，遵循项目的脚本规范
2. 设置@match规则匹配目标URL
3. 使用document-end确保页面加载完成后执行
4. 通过name属性精确定位邀请码输入框
5. 实现自动填写邀请码的核心逻辑

## 脚本结构
```javascript
// ==UserScript==
// @name         网站注册自动填写表单信息
// @namespace    https://getquicker.net/User/Actions/388875-星河城野❤
// @version      1.0
// @description  自动填写指定网站的注册表单信息
// @author       小张 | 个人博客：https://blog.z-l.top | 公众号"爱吃馍" | 设计导航站：https://dh.z-l.top | quicker账号昵称：星河城野❤
// @match        http://121.40.25.9:8080/register.html
// @grant        GM_addStyle
// @grant        GM_getValue
// @grant        GM_setValue
// @grant        GM_registerMenuCommand
// @icon         https://dh.z-l.top/assets/favicon.ico
// @run-at       document-end
// ==/UserScript==

(function() {
    'use strict';
    
    // 页面加载完成后执行
    document.addEventListener('DOMContentLoaded', function() {
        // 自动填写邀请码
        const inviteCodeInput = document.querySelector('input[name="inviteCode"]');
        if (inviteCodeInput) {
            inviteCodeInput.value = "1474728874";
            // 触发input事件，确保Layui表单验证能正常工作
            inviteCodeInput.dispatchEvent(new Event('input', { bubbles: true }));
        }
    });
})();
```

## 预期效果
- 打开目标注册页面时，邀请码输入框会自动填入"1474728874"
- 脚本执行后无明显延迟，不影响用户正常使用
- 触发input事件确保Layui表单验证能正常识别填写内容
- 遵循项目的代码规范和最佳实践
